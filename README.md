# igor-musin

Website in loving memory of Igor Musin.

Built with [Nuxt 4](https://nuxt.com) and [Nuxt Content](https://content.nuxt.com).

## Setup

Make sure to install dependencies using [pnpm](https://pnpm.io):

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:8110`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview the production build:

```bash
pnpm preview
```

## Lint

Run ESLint:

```bash
pnpm lint
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
